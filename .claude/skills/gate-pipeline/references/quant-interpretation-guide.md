## 3. Sharpe Ratio 분해

### 유형 분류

| 유형 | Sharpe | CAGR | Vol | 해석 |
|------|--------|------|-----|------|
| 고수익+고변동 | 1.5 | +80% | 50%+ | 강한 알파, 변동성 감내 필요 |
| 저수익+저변동 | 1.5 | +20% | 12% | 안정적이나 자본 효율 낮음 |
| 적정수익+적정변동 | 1.5 | +45% | 30% | **이상적** (vol-target 30-35%) |

### Red Flag: 비정상 Sharpe

| Sharpe | 해석 | 확인 사항 |
|--------|------|----------|
| > 3.0 | **과적합 강력 의심** | look-ahead bias, 비용 누락, 소수 거래 |
| > 2.5 | 과적합 의심 | G2 Decay 확인 필수 |
| 1.5-2.5 | 우수 (크립토 가능 범위) | 정상 범위 |
| 1.0-1.5 | 양호 | 배포 후보 |
| 0.5-1.0 | 보통 | 포트폴리오 분산용 |

## 4. IS/OOS Decay 패턴 (Gate 2)

### Decay 해석 표

| Decay | 등급 | 해석 | CTREND 비교 |
|-------|------|------|------------|
| < 20% | **우수** | 강한 일반화, 과적합 위험 낮음 | — |
| 20-35% | 양호 | 적절한 IS/OOS 갭 | CTREND: 33.7% |
| 35-50% | **경계** | 파라미터 의존성 주의 | 기준 상한 |
| > 50% | **과적합** | FAIL — 모델 단순화 필요 | — |

### Decay 음수 (OOS > IS) 해석

| 원인 | 가능성 | 확인 방법 |
|------|--------|----------|
| OOS 기간이 강세장 | 높음 | 레짐 달력 대조 |
| IS에 2022 약세장 포함 | 높음 | 데이터 분할 확인 |
| 코드 버그 | 낮음 | shift(1) 재검증 |

### G1 → G2 일관성

```
기대: G2 OOS Sharpe >= G1 Best Sharpe × 0.3

예시:
  G1 Best Sharpe = 2.05 → G2 OOS >= 0.615 기대
  CTREND 실제: 1.78 / 2.05 = 87% 유지 (우수)
```

## 5. 파라미터 고원 형태 (Gate 3)

### 고원 형태 분류

| 형태 | 그래프 | 해석 | 예시 |
|------|--------|------|------|
| **넓은 고원** | `___/‾‾‾‾‾‾\___` | **최적** — 로버스트 | CTREND alpha (0.1~1.0) |
| **좁은 봉우리** | `___/\___` | 과적합 — curve fitting | TTM bb_period (±2 급락) |
| **단조 경사** | `___/‾‾‾‾‾‾‾` | 방향성 존재 — 범위 확대 필요 | CTREND horizon (길수록 개선) |
| **절벽** | `‾‾‾‾‾\___` | 임계값 존재 — 안전 마진 확인 | TTM kc_mult 1.0 (거래 0건) |
| **U자형** | `\___/` | 양극단 우수 — 중간값 비효율적 | 드묾 |

### vol_target 특수 패턴

vol_target은 Sharpe에 거의 영향 없는 것이 **정상**:
- Sharpe ≈ 일정, CAGR만 증가 → 순수 레버리지 스케일링
- vol_target 변경 시 Sharpe도 크게 변하면 → 비선형 리스크 존재 (경고)

### 파라미터 간 교호작용 징후

| 징후 | 해석 |
|------|------|
| 파라미터 A 최적값이 파라미터 B 값에 따라 이동 | 강한 교호작용 — 2D 스윕 필요 |
| 두 파라미터 모두 같은 방향에서 좋음 (둘 다 크거나 작을 때) | 복합 효과 — 과적합 위험 |
| 각 파라미터가 독립적으로 고원 유지 | **이상적** — one-at-a-time 스윕 신뢰 가능 |

## 6. PBO 해석 (Gate 4)

### PBO + CPCV 복합 판정

| PBO | CPCV 전 fold OOS | 판정 | 근거 |
|-----|:-----------------:|------|------|
| < 40% | 전부 양수 | **강건** | 과적합 위험 낮고 전 fold 수익 |
| < 40% | 일부 음수 | 양호 | 과적합 낮으나 약한 fold 주시 |
| 40-60% | **전부 양수** | **경계 (CTREND 선례)** | PBO FAIL이나 실질 위험 제한적 |
| 40-60% | 일부 음수 | FAIL | 과적합 + 일부 fold 손실 |
| > 60% | 무관 | **FAIL** | 과적합 위험 높음 |

### CTREND 선례 상세

- PBO 60% > 40% FAIL
- 그러나 10개 CPCV fold **모두** OOS Sharpe 양수 (0.49~2.79)
- MC p-value 0.000, 95% CI [0.97, 1.87]
- **판단**: IS에서 최적이 OOS에서 차선이 되더라도 (순위 역전), 모든 fold에서 양수 수익이므로 실전 배포 시 방향성은 유지될 가능성 높음
- **다음 단계**: G5 EDA Parity에서 실시간 수준 검증

> 이 선례를 자동 적용하지 않는다. 각 전략에서 동일 조건 (전 fold OOS 양수 + MC PASS) 충족 여부를 개별 확인.

## 7. DSR n_trials 민감도 (Gate 4)

### n_trials 해석

| 구분 | n_trials | 의미 | CTREND |
|------|----------|------|--------|
| batch | 동일 배치 전략 수 (5-10) | 해당 연구 세션 내 다중 비교 보정 | 7 → DSR 1.00 |
| all | 전체 시도 전략 수 (20-50) | 프로젝트 전체 다중 비교 보정 | 31 → DSR 0.16 |

### 판정 매트릭스

| batch DSR | all DSR | 해석 |
|-----------|---------|------|
| > 0.95 | > 0.95 | **강건** — 전 프로젝트에서도 유의 |
| > 0.95 | < 0.95 | 양호 — 배치 내 유의, 전체 불확실 |
| < 0.95 | 무관 | **경고** — 배치 내에서도 불확실 |

> batch DSR > 0.95이 Gate 4 판정 기준. all DSR은 참고용.

## 10. Gate 간 일관성 패턴

### 건강한 파이프라인 패턴

```
G1 Sharpe 1.5 → G2 OOS 1.0 (Decay 33%) → G4 WFA OOS 0.8 (Decay 47%)
점진적 감소: 각 단계에서 20-40% 감쇠 = 정상적 일반화 손실
```

### 비건강한 패턴

```
G1 Sharpe 2.5 → G2 OOS 0.3 (Decay 88%) → G4 WFA OOS 0.1
급격한 감소: 과적합 전형. G1에서 이미 과적합된 상태.
```

```
G1 Sharpe 1.0 → G2 OOS 1.5 (Decay -50%) → G4 WFA OOS 0.2
비일관적: G2 역방향 + G4 급락 = 데이터 분할 인공물.
```

### 일관성 체크 표

| 비교 | 건강한 범위 | 경고 조건 |
|------|-----------|----------|
| G2 OOS / G1 Sharpe | 0.3 ~ 1.0 | < 0.3 (과적합) 또는 > 1.0 (의심) |
| G4 WFA OOS / G2 OOS | 0.5 ~ 1.2 | < 0.5 (window 의존) |
| G4 Decay - G2 Decay | -10% ~ +15% | 차이 > 20%p (방법론 민감) |
| G3 baseline Sharpe vs G1 Sharpe | 유사 (±10%) | 차이 > 20% (파라미터 변경) |
