## 레짐 커버리지 체크리스트

백테스트 기간이 다음 레짐을 **최소 1개씩** 포함하는지 확인:

- [ ] **강세장** (지속 상승, > +50%)
- [ ] **약세장** (지속 하락, > -40%)
- [ ] **횡보장** (레인지 바운드, 3개월+)
- [ ] **급락** (단기 -30% 이상)
- [ ] **급등** (단기 +30% 이상)

## 해석 시 주의사항

1. **2021 데이터만 사용** → 강세 편향 (결과 과대)
2. **2022 데이터만 사용** → 약세 편향 (MR 전략 과대)
3. **2023-2024만 사용** → 회복장 편향 (추세추종 과대)
4. **LUNA 크래시 포함** → MDD 극단값으로 왜곡 가능 (별도 분석 권장)
5. **BTC ≠ 알트** → BTC 레짐과 알트 레짐은 동기화 안 될 수 있음
