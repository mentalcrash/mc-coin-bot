### 포트폴리오 효과

```
분산 효과 = 1 - (Portfolio Sharpe / 평균 개별 Sharpe)

기대 분산 효과:
- 2-3 자산: 10-20% 개선
- 4-6 자산: 20-35% 개선
- 8+ 자산: 30-50% 개선 (감소 체감)
```

## 통계적 유의성 기준

| 거래 수 | 신뢰도 | 판단 |
|---------|--------|------|
| < 20 | 매우 낮음 | 참고만 |
| 20-50 | 낮음 | 조건부 해석 |
| 50-100 | 중간 | 해석 가능 |
| 100-300 | 높음 | 신뢰 가능 |
| > 300 | 매우 높음 | 강건한 결론 |

### Deflated Sharpe Ratio (DSR)

```
DSR = Sharpe - penalty(num_trials, skew, kurtosis)

해석:
- DSR > 0 + p-value < 0.05: 통계적 유의
- DSR > 0 + p-value > 0.05: 불확실
- DSR < 0: 우연에 의한 결과일 가능성
```
